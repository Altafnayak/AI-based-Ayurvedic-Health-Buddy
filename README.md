# 🏥 AI AyurVeda Hospital - Complete Healthcare System

## ✨ ALL FEATURES INCLUDED IN ONE PACKAGE

✅ **Patient Management** - Register & Login by Name
✅ **AI Diagnosis** - 10+ Diseases, Multiple Matches, Confidence Scores
✅ **Voice Assistant** - 🔊 Click to Hear Remedies Read Aloud
✅ **Health Tracking** - 7 Daily Metrics (HR, BP, Temp, Mental, Sleep, Exercise, Weight)
✅ **Health Graphs** - Visual Analytics (Line & Bar Charts)
✅ **Doctor Consultation** - Track All Doctor Visits
✅ **Complete Data Storage** - All Entries Saved Permanently
✅ **Auto-Load** - Past Data Loads When You Login

## 🚀 INSTALLATION (ONE-TIME)

### Terminal 1 - Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate      # Mac/Linux
pip install -r requirements.txt
python app.py
```

### Terminal 2 - Frontend  
```bash
cd frontend
npm install
npm start
```

**Visit: http://localhost:3000**

## 📋 WORKFLOW

1. **Register** - Create account with name, age, gender
2. **Login** - Enter your name → All past data auto-loads
3. **Dashboard** - See latest health metrics & stats
4. **New Diagnosis** - Describe symptoms → Get top 5 diseases with % confidence
5. **Voice Support** - Click 🔊 button → Hear remedies read aloud
6. **Track Health** - Log daily vitals (7 metrics) → Stored permanently
7. **View Graphs** - See health trends (Line & Bar Charts)
8. **Doctor Visits** - Add & manage doctor consultations
9. **History** - View all past diagnoses with voice support
10. **Ask veda** - AI agent which give remidies in local languge in kannda and in english with speak option
